# Overview

ALE Psych Community Docs is a community-maintained documentation site for ALE Psych Engine 5.

It is separate from the creator's existing documentation and is intended to stay focused on practical, easy-to-scan reference material.

## What to expect

- A clean home page with a live contributor list
- Local search for quick lookup
- Sidebar navigation for the site structure
- Edit links on pages so fixes stay simple
