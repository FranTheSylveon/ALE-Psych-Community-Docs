# Navigation

The site is organized into a small set of sections:

- `Home` for the landing page and contributor block
- `Guide` for the high-level project pages
- `Reference` for navigation and site usage notes

The default theme also shows:

- Local search
- Edit links
- Last-updated timestamps
- GitHub links in the top navigation and social area
